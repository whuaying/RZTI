# RZTI — Realization-Zoom Type Indicator

专为创客与极客设计的行为偏好分类工具。

MBTI 测的是你如何认知世界，RZTI 测的是**你如何造东西**。

## 四个维度

| 维度 | 左极 | 右极 | 含义 |
|------|------|------|------|
| **R** 实现路径 | M 数字派 (Modeler) | B 物理派 (Builder) | 你更倾向先建模推演，还是直接动手搭原型？ |
| **Z** 注意力焦点 | C 微观派 (Craftsman) | A 宏观派 (Architect) | 你更关注细节精度，还是系统架构？ |
| **T** 工作模式 | S 独行派 (Soloist) | O 团队派 (Organizer) | 你更享受独自深耕，还是群体协作？ |
| **I** 终极目标 | P 完美派 (Perfectionist) | F 务实派 (Fixer) | 你追求内部极致标准，还是外部实用价值？ |

## 16 种人格

4 个维度 × 2 极 = 16 种人格类型，每种有独特的能力画像、天然角色、真实痛点和最佳队友匹配。

## 测验结构

- 24 道迫选题（每个维度 6 道，覆盖 6 个生活/工作场景）
- 8 道程度题（每个维度 2 道，5 级李克特量表）
- 完成时间约 5 分钟

## 技术

纯静态 HTML 单文件，无后端依赖。打开 `index.html` 即可运行。
